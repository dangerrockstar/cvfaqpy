# CVFAQPy 🧠🤖

**A curated, interactive CLI tool for Computer Vision, Deep Learning, and Python interview preparation.**

[](https://www.python.org/)
[](https://opensource.org/licenses/MIT)

-----

## 📌 Overview

`CVFAQPy` is a Python-based interactive terminal application designed to help developers and AI enthusiasts master technical interviews. Instead of scrolling through endless PDFs, you can navigate through structured categories of frequently asked questions directly from your command line.

## 🚀 Key Features

  * **Interactive Menu:** Simple, number-based navigation system.
  * **Targeted Categories:**
      * **Computer Vision:** Image processing, filters, and geometry.
      * **Deep Learning:** Neural networks, backpropagation, and optimization.
      * **Python Essentials:** Language-specific logic and data structures.
  * **Zero Dependencies:** Runs on standard Python without needing external libraries like `OpenCV` or `PyTorch`.

-----

## 🛠️ Installation & Setup

To get started with `CVFAQPy` on your local machine:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/dangerrockstar/cvfaqpy.git
    ```

2.  **Navigate to the project folder:**

    ```bash
    cd cvfaqpy
    ```

3.  **Run the application:**

    ```bash
    python cvfaq.py
    ```

-----

## 📖 How to Use

Once you run the script, follow the on-screen prompts:

1.  **Main Menu:** Choose a category (e.g., Enter `1` for Computer Vision).
2.  **Question List:** A list of specific topics will appear. Select the number of the question you want to study.
3.  **Read & Learn:** The detailed answer will be displayed.
4.  **Navigate:** You can return to the main menu or exit the application at any time by following the menu instructions.
